Tutorial

Use LCLCam in OBS

  1. Create a connection and wait for the phone camera to appear in Studio.
  2. Copy the Camera URL from the camera card.
  3. In OBS, add a Browser Source and paste the Camera URL.
  4. Set Browser Source width and height to match the phone camera. For 1080p, use width 1920 and height 1080.
  5. Use OBS Virtual Camera if you want the phone camera inside Discord, Slack, Zoom, or similar apps.

Tutorial

Use LCLCam in Discord & Slack

  1. Open the camera in OBS using a Browser Source.
  2. Start OBS Virtual Camera.
  3. In Discord or Slack call settings, select OBS Virtual Camera.
  4. When you create a new LCLCam connection, update the Browser Source URL in OBS.
